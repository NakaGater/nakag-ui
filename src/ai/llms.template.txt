# @nakagater/ui {{VERSION}} — AI reference

shadcn/uiベースの個人デザインシステム。ビルド済みReactコンポーネント(Tailwind v4前提)と、同じデザイントークンを使うフレームワーク非依存の16:9スライドCSSを提供する。

## Wiring (required)

利用側のglobals.css / index.cssに次の3行が必要:

```css
@import "tailwindcss";
@import "@nakagater/ui/theme.css";
@source "../node_modules/@nakagater/ui/dist";
```

**@source行がないとコンポーネントのCSSが一切生成されない**(Tailwind v4はnode_modulesをスキャンしない。パスはCSSファイルからの相対)。ダークモードは `<html class="dark">` で切替。

## Importing

必ずルートからimportする。dist/へのdeep importは禁止。

```tsx
import { Button, Card, CardContent, cn } from "@nakagater/ui";
```

## Slides (pure HTML/CSS)

Reactもビルドも不要。`<section class="slide">`を`<body class="deck">`に並べるだけで16:9スライドになる。印刷→PDF保存で資料化。

```html
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/@nakagater/ui@{{VERSION}}/dist/slides/slide.css">
<!-- バージョンは必ず固定する。オフライン単一ファイルにするならslide.cssの中身を<style>に貼る -->
```
